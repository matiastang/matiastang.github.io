Github Secrets

约 195 字小于 1 分钟

2025-01-02

介绍

actions secrets相当于项目action中可以访问的配置项，在action中通过secrets.TOKEN访问，这样就不会把一些关键信息暴露出去。

配置

  1. 进入Github中需要配置的项目。
  2. 点击Settings进入项目配置。
  3. 展开左侧的Secrets and variables，选择Actions，后选择Secrets去添加secrets。
  4. 创建新的secrets，只需要填写Name和Secret即可。其中Name可以随便命名。

注意在action中通过secrets访问时，Name和创建的时候一致。如：Name设置为TOKEN则要使用secrets.TOKEN访问。
