HTML5

约 1956 字大约 7 分钟

2025-01-20

HTML5 是 HTML 的最新版本，它引入了许多新特性，旨在增强网页的功能和用户体验，同时减少对插件（如 Flash）的依赖，使得网页开发变得更加简单和强大。

HTML5 新特性：

HTML5 有以下十个新特性：

语义化标签

HTML5 引入了语义化标签，如 <header>、<footer>、<nav>、<section> 等，这些标签使得网页的结构更加清晰，易于理解和维护。更多的语义化的标签，使网页内容更具结构性和可读性，有助于 SEO 和可访问性（Accessibility）。

常见语义化标签：

  • <header>：定义页面或区块的头部
  • <footer>：定义页面或区块的底部
  • <nav>：定义导航菜单
  • <article>：定义独立的文章内容
  • <section>：定义内容的区域
  • <aside>：定义侧边内容
  • <figure> 和 <figcaption>：定义图像及说明

示例

<header>
  <h1>欢迎来到我的网站</h1>
</header>
<nav>
  <ul>
    <li><a href="#">主页</a></li>
    <li><a href="#">关于</a></li>
  </ul>
</nav>
<section>
  <h2>最新文章</h2>
  <article>
    <h3>HTML5的优势</h3>
    <p>HTML5提供了丰富的新特性...</p>
  </article>
</section>
<footer>版权所有</footer>

video和audio媒体元素

HTML5 引入了 <video> 和 <audio> 标签，使得在网页中嵌入视频和音频变得更加简单。无需依赖 Flash 插件。audio和video元素提供了方法、属性和事件，允许您使用 JavaScript 来操作audio和video元素，如：暂停、播放、音量条件等

audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  您的浏览器不支持 audio 元素。
</audio>

支持的格式：MP3、OGG、WAV

video

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  您的浏览器不支持 video 元素。
</video>

支持的格式：MP4（H.264）、WebM、Ogg Theora

表单控件

HTML5 引入了新的表单控件，增强了表单功能，提供了新的输入类型、属性和表单验证功能。如 <input type="date">、<input type="time">、<input type="email"> 等，使得表单的输入更加方便和直观。

<input type="email" placeholder="请输入邮箱">
<input type="number" min="1" max="10">
<input type="date">
<input type="range" min="0" max="100">

新增的表单元素

  • <datalist>：元素规定输入域的选项列表，使用<input>元素的list元素与<datalist>元素的id绑定
  • <keygen>：提供一种检验用户的可靠方法，标签规定用于表单的密钥对生成器字段
  • <output>：用于不用类型的输出，比如计算或脚本输出

新的表单属性

  • required（必填）：是一个boolean属性，要求填写的输入域不为空
  • pattern（正则表达式）：描述了一个正则表达式用于验证<input>元素的值
  • placeholder（占位符）：简短的提示在用户输入值前会显示在输入域上，既默认框提示
  • autofocus（自动聚焦）：是一个boolean属性，在页面加载时自动获得焦点
  • autocomplete（自动补全）：是一个boolean属性，规定输入字段是否应该启用自动完成功能
  • max/min：最大最小值
  • step：为输入域规定合法的数字间隔
  • height/width：用于image类型的<input>标签的图像高度和宽度
  • multiple：是一个boolean属性，规定<input>元素中选择多个值
<input type="text" required pattern="[A-Za-z]{3,}" title="至少输入3个字母">

Web 存储（localStorage & sessionStorage）

HTML5 提供了本地存储机制，取代了传统的 Cookies，可存储更大的数据量，数据仅在客户端存储。使得网页可以在用户的浏览器中保存数据，无需依赖服务器。

早些时候，本地存储使用的是cookies。但是Web存储需要更加的安全与快速，而且Web Storage中有5M容量，cookie只有4k。

localStorage（长期存储）

localStorage.setItem('username', 'John');
console.log(localStorage.getItem('username')); // 输出 John

sessionStorage（会话存储，浏览器关闭即清除）

sessionStorage.setItem('sessionID', 'abc123');
console.log(sessionStorage.getItem('sessionID'));

Canvas绘图

HTML5 引入了 Canvas 元素，使得网页可以绘制图形和动画。一般通过脚本操作 Canvas，通常是 JavaScript

<canvas id="myCanvas" width="400" height="200"></canvas>

<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');
  ctx.fillStyle = 'blue';
  ctx.fillRect(50, 50, 100, 100);
</script>

详细示例请查看Canvas

Web Socket

HTML5 引入了 WebSocket，使得网页可以与服务器进行全双工通信，实现实时数据交换。适用于实时应用，如聊天、股票行情。

WebSocket协议为web应用程序客户端和服务端之间提供了一种全双工通信机制。在WebSocket API中，浏览器和服务器只需要做一个握手的动作，然后浏览器和服务器之间就形成了一条快速通道，两者之间就直接可以数据互相传送。

浏览器通过JavaScript向服务器发出建立WebSocket连接的请求，连接建立以后，客户端和服务器端就可以通过TCP连接直接交换数据。

当你获取Web Socket连接后，你可以通过send()方法来向服务器发送数据，并通过onmessage事件来接收服务器返回的数据。

const socket = new WebSocket('ws://example.com/socket');
socket.onopen = () => socket.send('Hello Server!');
socket.onmessage = (event) => console.log('Message from server:', event.data);

Geolocation

HTML5 引入了 地理位置 API（Geolocation API），允许网页获取用户的地理位置信息，常用于地图和位置服务。

navigator.geolocation.getCurrentPosition((position) => {
  console.log('Latitude:', position.coords.latitude);
  console.log('Longitude:', position.coords.longitude);
});

完整使用示例请查看GeoLocation

注意 如果需要获取用户位置，需要用户授权，需要https协议。

Web Worker

HTML5 引入 Web Workers，允许在后台运行 JavaScript 线程，避免页面卡顿，提高性能。

当在HTML页面中执行脚本时，页面的状态是不可响应的，直到脚本已完成。Web Worker是运行在后台的JavaScript，独立于其他脚本，不会影响页面的性能。您可以继续做任何愿意做的事情：点击、选取内容等等，而此时Web Worker在后台运行。

const worker = new Worker('worker.js');
worker.postMessage('开始任务');
worker.onmessage = function(event) {
  console.log('Worker 结果:', event.data);
};

SVG 绘图

HTML5 支持嵌入矢量图形，使用 <svg> 元素创建可缩放的图形，适用于图标、图表等。

SVG 是指可伸缩的矢量图形，它是一种使用XML描述2D图形的语言。在SVG中，每个被绘制的图形均被视为对象。如果SVG对象的属性发生变化，那么浏览器能够自动重现图形。

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

提示 SVG 也可以与 JavaScript 一起使用，以创建交互式图形。

拖拽API

HTML5 提供了原生的拖放功能，拖放 API（Drag and Drop API）允许用户在网页中拖放元素。拖放是一种常见的特性，即抓取对象以后拖到另一个位置。

在HTML5中，拖放是标准的一部分，任何元素都能够拖放。拖放的过程分为源对象和目标对象。源对象是指你即将拖动元素，而目标对象则是指拖动之后要放置的目标位置。

拖放的源对象(可能发生移动的)可以触发的事件：

  • dragstart：拖动开始
  • drag：拖动中
  • dragend：拖动结束

整个拖动过程的组成：dragstart*1 + drag*n + dragend*1。

拖放的目标对象(不会发生移动)可以触发的事件：

  • dragenter：拖动着进入
  • dragover：拖动着悬停
  • dragleave：拖动着离开
  • drop：释放

整个拖动过程的组成：dragenter*1 + dragover*n + dragleave*1 或 dragenter*1 + dragover*n + drop*1。

<div draggable="true" ondragstart="drag(event)">拖我</div>

提示 这部分常鉴于可视化相关的项目。

新的事件和 API

  • 触摸事件（适用于移动设备）： touchstart、touchmove、touchend
  • 离线支持（Service Workers、Manifest 文件）
  • 全屏 API、剪贴板 API
