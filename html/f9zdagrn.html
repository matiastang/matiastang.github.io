p标签不可嵌入块级元素

约 701 字大约 2 分钟

2025-01-23

问题

在更新vuepress-theme-plume的copyright时使用了如下代码：

footer: {
    message: '...',
    copyright: '<div style="display: flex;flex-direction: column;align-items: center;justify-content: center;"><div>Copyright © 2022-Present <a href="mailto:emailaddress">tangdaoyong</a></div><div style="display: flex;align-items: center;justify-content: center;"><img src="/beian/beian-icon.png" style="margin: 0px 8px;" /><a href="https://beian.miit.gov.cn" target="_blank" rel="noopener noreferrer">ICP备号</a></div></div>',
},

vuepress-theme-plume的footer配置文档中，copyright是string类型的，所以这里使用了一个html字符串。

上面设置在本地测试的时候没有问题，但是部署到服务器上时，<div>标签被移除了，导致copyright内容没有显示出来。

查看页面的源码，发现打包后的代码如下：

<p class="copyright" data-v-d7f29ae6>
    <div style="display: flex;flex-direction: column;align-items: center;justify-content: center;">
        <div>
            Copyright © 2022-Present <a href="mailto:emailaddress">tangdaoyong</a>
        </div>
        <div style="display: flex;align-items: center;justify-content: center;">
            <img src="/beian/beian-icon.png" style="margin: 0px 8px;"/>
            <a href="https://beian.miit.gov.cn" target="_blank" rel="noopener noreferrer">ICP备号</a>
        </div>
    </div>
</p>

可以看出加载到页面内容里面是有这部分内容的。

页面解析出的页面如下：

<p class="copyright" data-v-d7f29ae6></p>

从上面的源码和解析出的代码可以看出，这里的copyright使用的是<p>标签，而<p>标签是不可以嵌入块级元素的，所以导致<div>标签被移除了，copyright内容没有显示出来。

为什么<div>会被移除？

一般我们都说<p>标签不可以嵌入块级元素，那为什么不能嵌入呢？

其实在 HTML5 解析规则中，遇到 <p> 标签，浏览器会遵循**自动闭合规则**，即在解析过程中，遇到另一个块级元素（如 <div>、<p>），浏览器会自动闭合外层的 <p> 标签标记会自动关闭前一个段落。

<p>This is a paragraph.
   <div>This is inside a div.</div>
</p>

如上的代码，浏览器解析后，会自动闭合前一个 <p> 标签，变成如下代码：

<p>This is a paragraph.</p>
<div>This is inside a div.</div>

解决方案

我们知道了copyright使用的是<p>标签，而<p>标签不能嵌入块级元素，所以我们更新配置为：

footer: {
    message: '...',
    copyright: '<span style="display: flex;flex-direction: column;align-items: center;justify-content: center;"><span>Copyright © 2022-Present <a href="mailto:emailaddress">tangdaoyong</a></span><span style="display: flex;align-items: center;justify-content: center;"><img src="/beian/beian-icon.png" style="margin: 0px 8px;" /><a href="https://beian.miit.gov.cn" target="_blank" rel="noopener noreferrer">ICP备号</a></span></span>',
},

如上使用span然后再通过style来调整样式，即可。
