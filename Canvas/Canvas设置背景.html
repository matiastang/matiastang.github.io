# canvas设置背景

当使用canvas绘制内容并将其保存为图片时，默认的背景颜色是透明的。因此有时需要主动设置一个背景颜色，以防止使用canvas保存的图片时显示的有问题。

canvas可以通过设置背景颜色或图片两种方式来处理上述问题。

# 设置背景颜色

canvas设置背景颜色分为两步：

  1. 获取到canvas的上下文
  2. 设置填充颜色
  3. 设置填充区域

获取到上下文后只需要下面两步就能初始化渲染区域的颜色。

// 设置填充样式
ctx.fillStyle = canvasConfig.bgColor
// 绘制矩形
ctx.fillRect(0, 0, width, height)

设置了渲染区域然后再在此基础绘制，然后保存的图片就具有背景了。

注意我们是为了设置canvas的背景，所以fillRect的宽高就应该是整个canvas的宽高。

# 微信小程序中 canvas 设置背景颜色

wx.createSelectorQuery()
    .select('#cavans') // canvas id
    .fields({ node: true, size: true })
    .exec((res) => {
        // canvas 对象
        const canvas: WechatMiniprogram.canvas = res[0].node

        // 渲染上下文
        const ctx = canvas.getContext('2d')

        // canvas 画布的实际绘制宽高
        const width = res[0].width
        const height = res[0].height

        // 初始化画布大小
        const dpr = wx.getWindowInfo().pixelRatio
        canvas.width = width * dpr
        canvas.height = height * dpr
        ctx.scale(dpr, dpr)

        // 设置填充样式
        ctx.fillStyle = '#f00' // 颜色值
        // 绘制矩形
        ctx.fillRect(0, 0, width, height)

        // 其他处理
   })

# H5 中的 canvas 设置背景图片

// 获取canvas
const canvas = document.getElementById('canvas') as HTMLcanvasElement

// 渲染上下文
const ctx = canvas.getContext('2d')

if (!ctx) {
  console.warn('canvas context is null')
  return
}

// 设置填充样式
ctx.fillStyle = '#f00' // 颜色值
// 绘制矩形
ctx.fillRect(0, 0, canvas.width, canvas.height)

# 设置背景图片

canvas设置背景图片分为两步：

  1. 获取到canvas的上下文
  2. 初始化背景图片
  3. canvas渲染图片

注意需要在 onload 回调中绘制到 canvas 中，已确保图片加载完成。可以在 onerror 回调中处理图片加载出错的问题。 注意通过 drawImage 设置 canvas 的背景时，如果图片的尺寸不是和 canvas 一样的话需要设置渲染的范围。

# 微信小程序中的canvas设置背景图片

wx.createSelectorQuery()
    .select('#cavans') // canvas id
    .fields({ node: true, size: true })
    .exec((res) => {
        // canvas 对象
        const canvas: WechatMiniprogram.canvas = res[0].node

        // 渲染上下文
        const ctx = canvas.getContext('2d')

        // canvas 画布的实际绘制宽高
        const width = res[0].width
        const height = res[0].height

        // 初始化画布大小
        const dpr = wx.getWindowInfo().pixelRatio
        canvas.width = width * dpr
        canvas.height = height * dpr
        ctx.scale(dpr, dpr)

        // 省略上面初始化步骤，已经获取到 canvas 对象和 ctx 渲染上下文
        // 图片对象
        const image = canvas.createImage()
        // 图片加载完成回调
        image.onload = () => {
            // 将图片绘制到 canvas 上
            // 保持图片原始大小
            ctx.drawImage(image, 0, 0)
            // 设置渲染的范围
            // ctx.drawImage(bgImg, 0, 0, canvas.width, canvas.height)
        }
        // 设置图片src
        image.src = 'bgimage url'

        // 其他处理
    })

获取到上下文后只需要下面两步就能将图片渲染到canvas。

// 图片对象
const image = canvas.createImage()
// 图片加载完成回调
image.onload = () => {
    // 将图片绘制到 canvas 上
    // 保持图片原始大小
    ctx.drawImage(image, 0, 0)
    // 设置渲染的范围
    // ctx.drawImage(bgImg, 0, 0, canvas.width, canvas.height)
}
// 设置图片src
image.src = 'bgimage url'

通过 canvas.createImage 创建图片对象并加载图片。当图片加载完成触发 onload 回调之后，使用 ctx.drawImage 即可将图片绘制到 canvas 上。

注意微信小程序中使用的是canvas.createImage创建图片对象。

# H5 中的 canvas 设置背景图片：

// 获取canvas
const canvas = document.getElementById('canvas') as HTMLcanvasElement

// 渲染上下文
const ctx = canvas.getContext('2d')

if (!ctx) {
  console.warn('canvas context is null')
  return
}

// 创建背景图片对象
const bgImg = new Image()

// 设置背景图片
bgImg.src ='bgImage url'

// 加载背景图片
bgImg.onload = () => {
   // 绘制背景图片
   // 保持图片原始大小
   ctx.drawImage(image, 0, 0)
   // 设置渲染的范围
   // ctx.drawImage(bgImg, 0, 0, canvas.width, canvas.height)
}
