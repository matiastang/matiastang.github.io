财务指标

每股收益（EPS，Earnings Per Share）

定义：公司净利润除以总股本
公式：EPS = 净利润 / 总股本
解读：

  • 基本EPS：仅考虑已发行股份
  • 稀释EPS：包含潜在稀释性证券（如可转债、期权等）
    作用：衡量公司盈利能力，EPS越高，每股创造的利润越多。

市盈率（P/E，Price-to-Earnings Ratio）

定义：股价与每股收益的比率
公式：市盈率 = 股价 / EPS
解读：

  • 静态市盈率：基于历史EPS
  • 动态市盈率：基于预测EPS
    作用：
  • 评估股票估值：高市盈率可能代表高成长预期或高估，低市盈率可能反映低估或增长乏力。
  • 需结合行业对比（科技股通常高于传统行业）。

每股净资产（BVPS，Book Value Per Share）

定义：净资产（总资产 - 总负债）除以总股本
公式：BVPS = 净资产 / 总股本
解读：反映每股对应的账面价值，代表股东权益的会计计量。

市净率（P/B，Price-to-Book Ratio）

定义：股价与每股净资产的比率
公式：市净率 = 股价 / BVPS
作用：

  • 价值投资参考：P/B < 1 可能被低估（资产清算价值高于股价）。
  • 适用行业：银行、保险等重资产行业更关注市净率。

ROE（净资产收益率，Return on Equity）

定义：净利润与净资产的比率
公式：ROE = 净利润 / 净资产 × 100%
解读：

  • 杜邦分析法分解：ROE = 净利润率 × 资产周转率 × 权益乘数
  • 高ROE：通常表明公司高效利用股东资金（如消费、科技行业ROE常超15%）。
    作用：衡量股东回报率，长期高ROE企业更具投资价值。

关键应用场景

  • 成长股：高EPS增速 + 高P/E（如科技公司）
  • 价值股：低P/B + 稳定ROE（如银行、制造业）
  • 风险提示：
    • 亏损企业无市盈率，轻资产公司（如互联网）市净率参考性低。
    • ROE过高可能因过度负债（需结合资产负债率分析）。

案例辅助理解

  • 公司A：股价20元，EPS=2元 → P/E=10倍（假设行业平均15倍，可能被低估）。
  • 公司B：BVPS=10元，股价15元 → P/B=1.5倍（若行业平均P/B=2倍，显示相对便宜）。
  • 公司C：ROE=25% → 若由高净利润率驱动（如茅台），质量优于靠高杠杆提升的ROE。

股票的最小交易单位

在中国A股市场，股票交易的最小交易单位（即“一手”）为100股，且买卖申报数量需为100股的整数倍（即1手、2手、3手……）。以下是详细规则和注意事项：

一、A股交易单位规则

  1. 主板/创业板/科创板

    • 最小买入单位：1手（100股）。
    • 最小卖出单位：不足100股需一次性卖出（如持有150股，可卖100股+50股，但50股需单独挂单）。
    • 申报数量：100股整数倍（如200股、500股）。

  2. 北交所

    • 最小交易单位：1手（100股）。
    • 申报数量：可1股递增（如101股、202股），但单笔申报最低100股。

  3. 港股（通过沪深港通）

    • 最小单位：由上市公司自行设定（常见为1手，可能是100股、500股等）。
    • 碎股：不足1手的股票可交易，但需通过特定渠道（如港股通的碎股卖出）。

二、特殊情况的交易规则

  1. 科创板/创业板

    • 最低交易金额：需超过200元人民币（防止低价股炒作）。
    • 示例：若股票价格1.5元/股，买1手需150元（低于200元），则无法申报。

  2. 退市整理期股票

    • 最小单位：1手（100股），但风险较高，需谨慎参与。

  3. 基金/债券

    • ETF基金：通常1手为100份。
    • 国债逆回购：沪市10万元起，深市1000元起。

三、常见问题解答

  1. 能否买少于100股？

    • A股：不可主动买入，但分红送股可能导致持仓不足100股（可一次性卖出）。
    • 北交所：可买入101股等非整数倍，但需满足最低100股。

  2. 高价股交易门槛

    • 示例：贵州茅台股价1700元/股，1手需17万元，资金门槛较高。

  3. 港股与A股差异

    • 港股每手股数不固定（如腾讯1手100股，汇丰1手400股），需具体查询。

提示：交易规则可能随政策调整，建议以交易所最新公告为准（参考上交所、深交所官网）。

如何查看一只股票是否在某个指数中

方法1：通过指数公司官网查询（最权威）

如访问中证指数有限公司官网，网址：www.csindex.com.cn

要判断一只股票是否属于某个特定指数（如沪深300指数），可以通过以下几种方法进行查询和验证：

方法1：通过指数公司官网查询（最权威）

  1. 访问中证指数有限公司官网
    • 网址：www.csindex.com.cn
  2. 搜索目标指数
    • 在搜索栏输入“沪深300”或指数代码“000300”，进入指数详情页。
  3. 下载成分股列表
    • 在指数页面找到“成分股”或“下载”选项，获取最新的Excel或PDF格式的成分股名单。
  4. 按股票代码或名称搜索
    • 使用快捷键（Ctrl+F）输入股票代码或名称，确认是否在列表中。

方法2：通过金融数据平台查询

同花顺/东方财富/雪球等软件

  1. 输入股票代码或名称，进入个股详情页。
  2. 在“公司概况”或“F10资料”中查找“所属指数”栏目（部分软件显示为“指数成分”）。
  3. 若显示“沪深300”或其他指数，即表示属于该指数成分股。

专业终端（Wind/Choice）

  • 使用函数或筛选工具：
    • Wind：输入股票代码后，在“指数成分”模块查询。
    • Choice：通过“指数分析”功能筛选。

方法3：通过券商或财经网站

  1. 新浪财经/东方财富网
    • 搜索“沪深300成分股”，查看最新名单。
  2. 券商研报
    • 部分券商会在指数调整时发布成分股变动分析报告（如中金、中信等）。

方法4：直接查看股票代码规则（辅助判断）

  • 沪深300成分股代码范围：
    • 沪市：60/68开头（如贵州茅台600519）。
    • 深市：00/30开头（如宁德时代300750）。
  • 例外：并非所有60/68/00/30开头的股票都是沪深300成分股，需结合其他方法确认。

注意事项

  1. 指数定期调整
    • 沪深300成分股每半年调整一次（6月和12月），需确认查询的是最新名单。
  2. 临时调整
    • 若成分股被ST、退市或发生重大风险，可能会被提前剔除。
  3. 跨市场指数
    • 沪深300包含沪深两市的股票，而类似上证50仅限沪市股票。

示例验证（以贵州茅台为例）

  1. 官网查询：
    • 在中证指数官网下载沪深300成分股列表，搜索“600519”或“贵州茅台”，确认存在。
  2. 软件查询：
    • 在同花顺中输入“600519”，F10资料显示“所属指数：沪深300”。

通过以上方法，您可以快速准确地验证某只股票是否属于目标指数。
