windows安装并使用git

约 370 字大约 1 分钟

Windowsgit

2025-01-08

介绍

git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

安装

Windows安装

在命令行查看git版本，确认本机是否安装了git：

$ git --version
git version 2.46.0.windows.1

如果没有安装git先安装。直接在git网站上下载最新的git版本安装即可。

如果已经安装有git，也可以选择更新git到最新版本：

$ git update-git-for-windows
Git for Windows 2.46.0.windows.1 (64-bit)
Update 2.47.1.windows.1 is available
curl: (35) schannel: next InitializeSecurityContext failed: CRYPT_E_REVOCATION_OFFLINE (0x80092013) -

这个curl一般都是网络的问题，可能是我开了代理的问题，关掉本地代理再试。

$ git update-git-for-windows
Git for Windows 2.46.0.windows.1 (64-bit)
Update 2.47.1.windows.1 is available
curl: (35) schannel: next InitializeSecurityContext failed: CRYPT_E_REVOCATION_OFFLINE (0x80092013) -
C:\Users\matia>git update-git-for-windows
Git for Windows 2.46.0.windows.1 (64-bit)
Update 2.47.1.windows.1 is available
Download and install Git for Windows v2.47.1.windows.1 [N/y]? y
################################################################################################################ 100.0%

下载完成后，会自动弹出更新提示弹窗，点击确定就开始更新。

提示 下载过程很缓慢，不知道是从哪儿下载的，不知道能加速不。

更新完成后，再次查看git版本

$ git --version
git version 2.47.1.windows.1

注意 前面所说的的命令行使用的是Windows PowerShell。
