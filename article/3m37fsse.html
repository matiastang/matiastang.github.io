Homebrew

约 230 字小于 1 分钟

macoshomebrew

2025-01-11

介绍

Homebrew主要用来安装 Apple（或您的 Linux 系统）没有预装的一些软件。

安装

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

使用

关闭自动更新

要关闭或跳过 Homebrew 的自动更新行为，可以使用以下几种方式：

临时跳过自动更新

HOMEBREW_NO_AUTO_UPDATE=1 brew install <包名>

示例：

HOMEBREW_NO_AUTO_UPDATE=1 brew install --cask mactex

永久关闭自动更新

在 shell 配置文件（如 .zshrc 或 .bashrc）中添加如下变量：

# Homebrew config start
# 禁用 Homebrew 的自动更新
HOMEBREW_NO_AUTO_UPDATE=1
# Homebrew config end

对于 Bash 用户，请替换为 .bashrc 或 .bash_profile

提示

关闭自动更新后，Homebrew 不会再在每次命令执行前更新仓库，但你仍可以手动更新： sh $ brew update
