# MySQL修改密码

# 1. 使用 SET PASSWORD 命令

  • 登录

输入命令mysql -u root -p指定 root 用户登录 MySQL，输入后按回车键输入密码。如果没有配置环境变量，请在 MySQL 的 bin 目录下登录操作。

  • 使用 SET PASSWORD 修改密码

命令格式为 set password for username@localhost = password(newpwd);，其中 username 为要修改密码的用户名，newpwd 为新密码。

  • 输入quit;命令退出 MySQL 重新登录，输入新密码root登录就可以了。

# 2. 使用mysqladmin修改密码

使用 mysqladmin 命令用户密码格式为：mysqladmin -u用户名 -p旧密码 password 新密码。

# 3. UPDATE直接编辑user表

  • 指定 root 用户登录 MySQL：mysql -u root -p

  • 也可以配置skip-grant-tables参数，跳过授权表。并使用sudo mysql直接使用root方式登录，不需要指定用户名密码。

首次配置时，可以在配置文件中设置skip-grant-tables参数，该命令作用是跳过授权表，就是说谁都能进入mysql看到所有数据表，输入任意字符账号密码都可以，当忘记账号密码时可以使用该命令修改密码，但是要随用随关，重启mysql，不然服务器上会有很大的风险。

# 查看密码

mysql> select user,host,password from user;
ERROR 1054 (42S22): Unknown column 'password' in 'field list'

因为mysql-5.7及以上版本已没有password字段，已将password字段修改为authentication_string字段

将password修改为authentication_string

update user set authentication_string=password('you new password') where user='root';
