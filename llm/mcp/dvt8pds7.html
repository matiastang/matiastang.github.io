VSCode + Cline

cline

再VSCode中安装Cline插件，然后选择服务提供商，比如：OpenRouter点击获取API KEY回跳转到OpenRouter页面的登录页面，这里我选择的是使用Chrome账号登录，然后设置了Key跳转回VSCode中，完成配置。模型选择的是deepseek/deepseek-chat:free。

这个Cline还会显示Task的使用数据，比如tokens和上下文窗口占用等

OpenRouter中可以使用免费的deepseek/deepseek-chat:free模型。

Cline支持配置多种API key，包括Open Router、Open AI和Ollama等。要使用免费模型，可以访问glfh.chat网站，该网站提供了多个免费的开源模型，包括：

llama3.1 405B 千问2.5 coder 32B llama3.3 70B

数据对话提示成功后就表示配置好了。接下来我们需要安装一些MCP Server，可以在Cline的插件中搜索服务。

Cursor

smithery.ai 。这是一个MCP Server共享平台，用户可以在上面发布和下载MCP Server配置。在这里可以选择你需要的MCP 服务。

注意 这里需要登录之后才可查看配置，直接使用github账号登录即可。

在Cursor Setting中选择MCP中点击Add new Global MCP server按钮，将选择的服务配置添加到配置文件中即可。

{
    "mcpServers": {
      "mcp-weibohot-server": {
        "command": "npx",
        "args": [
          "-y",
          "@smithery/cli@latest",
          "run",
          "@wangtsiao/mcp-weibohot-server",
          "--config",
          "{}"
        ]
      }
    }
}

添加完成之后再MCP Servers中，看见服务名称前中的小点儿成绿色就表示添加上了。

提示 第一次添加失败了，刷新之后好了。
